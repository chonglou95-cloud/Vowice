# Vowice 一周 MVP 与产品经理作品集计划

## 目标

在 7 天内，由 1 名产品经理与 1 名开发者，每人每天投入 4–5 小时，完成可运行的 Vowice MVP、可交互原型、PRD 与初步用户调研，并沉淀为产品经理求职作品集材料。

## 核心约束

- 不自行训练大模型或声音模型。
- 不使用人工配音作为 MVP 依赖。
- 技术方案必须支持中文语音对话，并优先考虑许可证、延迟、成本与部署难度。
- 本周交付的是可演示 MVP，不是 App Store 正式上架版。
- 产品效果优先于过度压缩功能；通过 Codex 与 Claude Agent 执行夜间开发、测试和文档工作，增加一周内可实现的体验完整度。
