# 声契 Vowice

> 听见他，也被他记住。  
> Hear him. Be remembered.

Vowice 是一款以声音为核心的 AI 陪伴产品，面向希望获得长期、可控且具有亲密关系连续性感受的用户。本仓库用于统一管理用户研究、产品文档、原型设计、MVP 开发。

## 产品定位

- 中文名：声契
- 英文名：Vowice（Vow + Voice）
- 品牌含义：由声音承载的承诺
- 产品主张：一段存在于声音里的记忆
